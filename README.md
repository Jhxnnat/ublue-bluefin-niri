This is a bluefin customized image that I'm workign on: bluefin with all the necessary requirements to be able to run my niri configuration alongside gnome.

- [Bluefin](https://projectbluefin.io/) used as the base image.
- Pre installed list of Flatpaks and Brews
- The niri configuration from my [dotfiles](https://github.com/jhxnnat/dotfiles)
